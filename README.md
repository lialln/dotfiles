### Install

1. Clone the repo

 `git clone https://github.com/lxyad/dots.git ~/.dotfiles`

2. Enter the new cloned repo

 `cd ~/.dotfiles`

3. Install dependencies & bootstrap the installation

 a)`script/install`
 b)`script/go`
